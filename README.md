# 概要
練習室の予約アプリです。ユーザーは、[管理者①][管理者②]　　[一般利用者]　　の３つに権限が分かれています。

# 使い方

トップ画面からユーザー登録/ログインをしてください　　

[管理者①]

・管理者②を追加/管理することができます　　

[管理者②]　　

・施設情報の追加/管理ができます　　

・施設の予約状況を確認することができます　　

・施設の予約ができます　　

・自分の予約履歴の確認/キャンセルができます　　

[一般利用者]　　

・施設の予約ができます　　

・自分の予約履歴の確認/キャンセルができます
